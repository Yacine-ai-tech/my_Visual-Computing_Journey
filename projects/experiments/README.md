# Experiments

Experimental code and parameter testing scripts.

## Contents

- `threshold_tests.py` - Comparison of thresholding methods (simple, Otsu's, adaptive)
- `blur_kernel_tests.py` - Blur kernel size effects on edge detection
- `colorspace_tests.py` - Color space transformation tests
- `feature_detection_wip.py` - Feature detection experiments (work in progress)

## Purpose

This directory contains:
- Parameter tuning and testing
- Algorithm comparisons
- Prototype implementations
- Quick tests and experiments

Note: Code in this directory is experimental and may not be production-ready.
